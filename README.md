# 🌐 BlueFolio — Creative Portfolio & Moderation Platform

**BlueFolio** เป็นแพลตฟอร์มเว็บแอปพลิเคชันสุดพรีเมียมสำหรับการจัดแสดงผลงานสร้างสรรค์ (Portfolio Showcasing) ที่ออกแบบมาเพื่อครีเอเตอร์ในการนำเสนอผลงานประเภทรูปภาพ วิดีโอ หรือเอกสาร พร้อมระบบรักษาความปลอดภัยและแผงควบคุม Moderation แบ่งระดับผู้ดูแลระบบและครีเอเตอร์อย่างเต็มรูปแบบ

---

## 🚀 ฟังก์ชันหลักของระบบ (Core Features)

### 👥 1. ระบบยืนยันตัวตนและการเข้าถึง (Role-Based Authentication)
* **การลงทะเบียน & เข้าสู่ระบบ**: บันทึกข้อมูลแบบ Persistent ผ่าน `localStorage` พร้อมการสุ่มและออกรหัสผู้ใช้จำเพาะ เช่น `ADM01` (แอดมิน) หรือ `USR01` (ครีเอเตอร์)
* **การรักษาหน้าจอหลังรีเฟรช (Refresh Persistence)**: จดจำแท็บและหน้าที่เปิดค้างไว้ล่าสุดก่อนกดรีเฟรชหน้าจอ เพื่อไม่ให้ผู้ใช้วิ่งกลับไปหน้าฟีดหลักทุกครั้ง
* **การแยกบทบาทการเข้าถึง (RBAC)**: แบ่งบัญชีผู้ใช้เป็น 3 ระดับ ได้แก่ **Super Admin**, **Admin**, และ **Creator** เพื่อควบคุมสิทธิ์การแสดงผลและระบบการจัดการ

### 🛡️ 2. แผงควบคุมผู้ดูแลระบบแบบ 2 ระดับ (Two-Level Admin Console)
แท็บ Moderation จะแสดงผลตามระดับสิทธิ์ของแอดมินที่ล็อกอินเข้ามา:
* **ระดับสูง (Super Admin)**: เข้าถึงการจัดการครบ 3 ส่วนหลัก:
  * **จัดการครีเอเตอร์**: ระงับ/ปลดระงับการโพสต์ของสมาชิกคนใดคนหนึ่ง หรือลบบัญชีครีเอเตอร์ทิ้งถาวร (ลบโพสต์ของคนนั้นทั้งหมดแบบ Cascade)
  * **จัดการแอดมิน**: ตรวจสอบรายชื่อผู้ดูแลระบบทั่วไป, เพิ่มบัญชีผู้ดูแลระบบคนใหม่ (Create Admin) และกดลบแอดมินปกติออกได้
  * **จัดการโพสต์ผลงาน**: กดซ่อน (ระงับโพสต์) หรือกดลบโพสต์ผลงานเดี่ยวใดๆ ออกจากระบบ
* **ระดับปกติ (Admin)**: 
  * ดูแลหน้าตาราง **จัดการโพสต์ผลงาน** เพียงแท็บเดียวเท่านั้น (ซ่อนแท็บแอดมินและครีเอเตอร์โดยสิ้นเชิงเพื่อความปลอดภัย)
  * ปฏิบัติงานคัดกรองเนื้อหา ระงับหรือลบโพสต์ผลงานที่ไม่เหมาะสมได้
* **การซิงก์ผลการค้นหา (Admin Table Search)**: ตารางแอดมินในแต่ละแท็บสามารถกรองข้อมูลได้แบบเรียลไทม์ตามคำค้นหาในช่องค้นหาหลักเพื่อความสะดวกในการจัดการ

### 🎨 3. พื้นที่ทำงานของครีเอเตอร์ (Creator Workspace & Profile)
* **อัปโหลดผลงาน (รูปภาพ/วิดีโอ)**: รองรับไฟล์รูปภาพ (JPG/PNG) และวิดีโอ (MP4) พร้อมระบบพรีวิว (ลบตัวเลือกประเภทไฟล์เอกสาร PDF/Docs ออกจากระบบโดยสมบูรณ์)
* **การแก้ไขข้อมูลแบบจำกัด (Restricted Post Edits)**: โพสต์ที่อัปโหลดไปแล้วระบบจะอนุญาตให้แก้ไขได้เฉพาะ **ชื่อผลงาน**, **รายละเอียด**, และ **แท็ก** เท่านั้น โดยข้อมูลไฟล์มีเดียและประเภทโพสต์จะถูกล็อคไว้อย่างปลอดภัย
* **การตรวจสอบรูปแบบแฮชแท็ก (Tag Validation)**: การพิมพ์แท็กต้องขึ้นต้นด้วย `#` และไม่มีเว้นวรรค เช่น `#Design #React` โดยระบบจะแจ้งเตือนผ่าน Rose Alert Banner ทันทีหากพิมพ์รูปแบบไม่ถูกต้อง
* **ประวัติย่อและลิงก์เชื่อมโยง (Bio & Social Links)**: ปรับปรุงส่วนแก้ไขโปรไฟล์ให้สามารถกรอก Bio และ Social Link ได้ โดยจะแสดงผลบนโปรไฟล์และบนการ์ดหน้าแสดงรายละเอียดผลงาน
* **Visitor Mode**: ปุ่มแสดงตัวอย่างหน้าโปรไฟล์คลังผลงานของตัวเองในมุมมองผู้เยี่ยมชมสาธารณะ
* **การ์ดต้อนรับสำหรับผู้เยี่ยมชม (Guest Workspace CTA)**: สำหรับผู้เยี่ยมชมที่ยังไม่ได้ลงชื่อเข้าใช้งาน เมื่อคลิกที่แท็บ "จัดการผลงาน" ระบบจะแสดงการ์ดพรีวิวพร้อมปุ่มให้กดเปิดหน้าต่างเข้าสู่ระบบผ่าน hash `#login`

### 🔍 4. ระบบฟีดสำรวจและการค้นหา (Explore Feed & Smart Search)
* **ระบบนำทางแบบ URL Hash Routing (Browser Navigation support)**: ผูกสถานะแท็บผลงาน (`#workspace`, `#admin`, `#login`), ครีเอเตอร์ที่เลือกกรอง (`#creator/อีเมล`), ตัวกรองแท็ก (`#tag/ชื่อแท็ก`), และหน้าต่างรายละเอียดผลงาน (`#project/ไอดี`) เข้ากับ URL Hash เพื่อให้ผู้ใช้สามารถกด **ปุ่มย้อนกลับ (Back Button) ของเบราว์เซอร์** เพื่อนำทางในระบบได้อย่างเป็นธรรมชาติ
* **ตัวกรองผลงานขั้นสูงในรูปแบบป๊อปอัป (Popup Filter Window)**: ปรับปรุงแถบหมวดหมู่หลัก (ทั้งหมด, รูปภาพ, วิดีโอ) ให้เรียบหรูอยู่ด้านบนสุด และมีปุ่มเปิดหน้าต่างป๊อปอัป "ตัวกรองผลงานขั้นสูง" เพื่อเลือกประเภทและแท็กทั้งหมดในระบบ
* **ปุ่มคัดกรองแท็กยอดนิยมด่วน (Popular Tags Quick Filters)**: นำแท็กที่ถูกใช้โพสต์เยอะที่สุดมาแสดงด้านบน Explore Feed ช่วยให้ผู้ใช้กดฟิลเตอร์งานตามแท็กยอดนิยมได้ทันทีในคลิกเดียว
* **ช่องค้นหาหลักเด่นชัดบน Navbar (Prominent Search Bar)**:
  * ปรับโฉมช่องค้นหาให้มีขนาดใหญ่ขึ้น ขอบหนา และมีเงาเรืองแสงสีน้ำเงินที่ทันสมัย
  * ค้นหาแท็กแบบตรงจุดเมื่อพิมพ์เครื่องหมาย `#` (เช่น `#Design`) และแสดงรายการแนะนำแฮชแท็ก (Tag Autocomplete Dropdown) อัจฉริยะขณะกำลังพิมพ์
  * รองรับการพิมพ์คำค้นหาแยกหลายคำพร้อมกัน (Multi-term search)
* **ระบบกดถูกใจ (Like System)**: แสดงปุ่มรูปหัวใจและตัวเลขนับจำนวนไลก์ทั้งบนการ์ดผลงานและในหน้าต่างแสดงรายละเอียดผลงาน เพื่อเพิ่มความสามารถในการโต้ตอบของระบบ
* **ระบบติดตามและคัดกรองฟีดติดตาม (Follow System & Following Feed)**: สามารถกดติดตามผู้สร้างชิ้นงานที่ต้องการได้ผ่านโมดอลรายละเอียด และกดเปิดฟิลเตอร์ "เฉพาะที่ติดตาม" เพื่อดูเฉพาะผลงานของครีเอเตอร์ที่เรากดติดตามไว้
* **ระบบความคิดเห็นและแก้ไขปัญหาทับซ้อน (Comments Overlap Fix)**: เปิดให้แสดงความคิดเห็นได้บนโมดอลรายละเอียดผลงาน โดยปรับสไตล์ปุ่มลบคอมเมนต์และวันที่ให้อยู่ข้างกันแบบ Inline เพื่อไม่ให้เกิดปัญหารอยอักษรหรือปุ่มลบทับกันแบบ absolute ของเดิม

## 🛠️ เทคโนโลยีที่ใช้พัฒนา (Tech Stack)

* **แกนหลัก (Core)**: [React.js](https://react.dev/) + [Vite](https://vite.dev/) (ชุดพัฒนาความเร็วสูงรองรับ HMR)
* **การตกแต่งหน้าตา (Styling)**: [Tailwind CSS v4](https://tailwindcss.com/) (การจัดวางสไตล์ แอนิเมชัน และ Grid สุดพรีเมียม)
* **ไอคอน (Icons)**: [Lucide React](https://lucide.dev/)
* **การจัดเก็บข้อมูล (Storage)**: Local Storage API ฝั่ง Client (ซิงก์ข้อมูลไม่สูญหายเมื่อรีเฟรชเบราว์เซอร์)

## 💻 วิธีการดาวน์โหลดและเริ่มใช้งาน (Getting Started)

### 1. ติดตั้ง Dependencies ในโฟลเดอร์โครงการ:
```bash
npm install
```

### 2. รันระบบสำหรับพัฒนา (Local Development Server):
```bash
npm run dev
```
เปิดบราวเซอร์ไปที่ลิงก์ระบบ (เช่น `http://localhost:5173`) เพื่อเริ่มทดลองใช้งาน

### 3. ตรวจสอบการคอมไพล์และคอมไพล์สำหรับ Production (Build Project):
```bash
npm run build
```
ไฟล์สำหรับขึ้นเซิร์ฟเวอร์จริงจะถูกสร้างไว้ในโฟลเดอร์ `dist/`
